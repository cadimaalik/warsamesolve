How to Contribute

STEP 1: Choose Your Topic

Pick a course, chapter, or specific problem you want to automate. It can be anything from CE 372 Hydromechanics to structures, transportation, geotechnical — any calculation that wastes students' time.

STEP 2: Write Your Code

Submit in any format that works for you:

  • Raw code: Python, C++, MATLAB, JavaScript, or any language
  • Pseudocode: Step-by-step logic in plain language
  • Language: Turkish or English — farketmez
What is pseudocode? Step-by-step instructions in plain language that describe how to solve a problem, without actual programming syntax.

Example:
1. Get pipe diameter and flow rate from user
2. Calculate velocity = flow rate / area
3. Calculate Reynolds number = velocity × diameter / viscosity
4. If Reynolds > 4000, use turbulent flow formula
5. Display results

What We're Looking For

Any engineering calculation that steals hours from students:

  • Hydromechanics: Open channel flow, pump systems, pipe networks
  • Numeric Methods: Root finding, integration, differential equations
  • Transportation: Traffic flow analysis, level of service, capacity
  • Structures: Moment distribution, truss analysis, beam design
  • Geotechnical: Bearing capacity, settlement, slope stability
  • Environmental: Water treatment, air quality calculations
  • Any ODTÜ CE course: If it involves repetitive calculations, we want it

Remember: "Coming soon" doesn't mean nothing. If you have an idea, share it — don't wait for someone else to do it.
